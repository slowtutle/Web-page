HTML

지금부터 여기서 적히는 코드는 <>를 대신하여 ()를 사용하며 지금까지 배운 코드를 설명한다

(!doctype html)
웹 페이지를 만드는 과정에서 가장 위쪽에 입력하는 코드이다

(html)
(!doctype html) 바로 다음으로 입력되어지는 언어이다

(head)
코드에 머리쪽이고 생각하면된다 예로 (meta charset="utf-8"), (title) 등이 여기에 속하고 있다

(body)
코드에 몸통쪽이라고 생각하면된다 제목, 목록, 본문 등이 여기에 속하고 있다

(h1 ~ h6)
제목, 숫자가 낮아질수록 글자에 크기가 커지며 숫자가 커질수록 글자가 작아지는 코드 h6이 가장 큰 숫자이다

(ol), (ul), (li)
(ol), (ul) 같은 경우는 (li)를 따라 함께 따라입력해야한다 주로 목록을 입력하는데 사용되는 코드이다, (ol) 목록앞에 숫자를 자동으로 기입해주는 코드, (ul) 숫자 기입을 원하지 않을때 사용되는 코드, 이 두 코드 중 하나는 반드시 입력하고 다음줄에 (li)목록(/li) 이런 형태로 입력하여야한다

(p)
본문, 제목 다음에 입력되는 코드이다

(br)
줄 바꾸기, 글 작성도중 줄을 바꿀때 사용되는 코드이다

(u)
밑줄 긋기, 글 작성도중 원하는 글에 밑줄을 그을때 사용되는 코드이다

(strong)
글 강조, 글 작성도중 원하는 글에 굵은 글씨로 작성할때 사용되는 코드이다

(img)
사진을 첨부하고자 할때 사용되는 코드 (img src="첨부하고자 하는 이미지" width="이미지 크기%") 이미지 크기 입력시 100% 입력시 화면에 맞게 크기가 맞춰진다

(a)
글 클릭시 지정한 링크로 이동 시켜주는 코드 (a href="이동시키고자 하는 링크" target="_blank" title="이동하는 링크 설명") 여기서 target="_blank"는 새로운 인터넷창을 열어서 이동시켜주는 코드 입력하지 않을시 현재 있는 인터넷창에서 바로 이동시켜준다
